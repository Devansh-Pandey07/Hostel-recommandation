VIT Hostel Recommendation System

This project predicts the best hostel blocks for students based on:

- CGPA
- Bed type preference
- AC / Non-AC preference
- Available beds in each block

The system uses a filtering algorithm and a heap-based ranking method written in C.

Project Files

filter.c       -> Filters hostels based on preferences
heap.c         -> Heap algorithm to rank hostel options
room.h         -> Struct definition for hostel data
hosteldata.txt -> Dataset of hostel blocks and bed availability
index.html     -> Prototype website interface

Prototype Website

The project also includes a simple frontend that simulates the hostel recommendation system.
